# Controllers

This folder contains the request handlers and route middleware for the API.

## Files

- `UserController.ts` - user CRUD handlers plus id validation middleware.
- `CategoryController.ts` - category CRUD handlers plus id validation middleware.
- `ProductController.ts` - product CRUD handlers plus id validation middleware with enhanced category handling (accepts both ObjectId and category name).
- `OrderController.ts` - order CRUD handlers plus id validation middleware.
